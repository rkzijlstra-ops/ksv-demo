# Demo-project Keukenstudio Voorschoten vastgelegd

Datum: 2026-05-24

## Wat is besloten

Mini-demo voor eerste gesprek met Ed:
- Monteur-input + PDF-input + samengevoegde lijst
- Stack: Next.js + Supabase + Vercel
- App-vorm: PWA (Progressive Web App), geen native app
- Vanaf nul in 01_projecten/keukenstudio-voorschoten-demo/
- 11-19 uur bouwtijd, 3-4 sessies

## Belangrijkste keuzes

- Stack-keuze met focus op waar Claude Code het sterkst in is, niet op wat ik ken
- PWA in plaats van native: vermijdt dubbel werk en App Store-gedoe, voor MKB-use-case ruim voldoende
- Schone start, geen fork van trc-platform
- Demo dient twee doelen: Ed overtuigen + 70-80% code herbruikbaar voor echte versie
